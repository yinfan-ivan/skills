### 输入规格
| 类型 | 格式 | 数量 | 大小限制 |
|------|------|------|----------|
| 图片 | jpeg/png/webp/bmp/tiff/gif | ≤ 9 张 | < 30 MB |
| 视频 | mp4/mov | ≤ 3 个，总时长 2-15s | < 50 MB |
| 音频 | mp3/wav | ≤ 3 个，总时长 ≤ 15s | < 15 MB |
| 文本 | 自然语言 | - | - |

**混合输入总上限：12 个文件**

## 🚀 火山方舟体验中心操作指南

### 访问入口
**官方地址**：https://console.volcengine.com/ark/region:ark+cn-beijing/experience/vision?modelId=doubao-seedance-2-0-260128

### 操作步骤

#### 第一步：进入体验中心
1. 打开火山引擎官网，登录账号
2. 进入「火山方舟」控制台
3. 点击左侧「体验中心」→ 选择「视频生成」
4. 确认模型选择为 **Seedance 2.0**

#### 第二步：上传素材（可选）
1. 点击「上传素材」按钮
2. 选择参考图片/视频/音频文件
3. 系统会自动命名（如 @图片1、@视频1）

#### 第三步：输入提示词
1. 在文本框中输入结构化提示词
2. 使用 @素材名 引用上传的文件
3. 可勾选「使用首帧」等功能选项

#### 第四步：选择参数
- **生成时长**：4s / 8s / 15s（视账户额度而定）
- **视频比例**：竖屏(9:16) / 横屏(16:9) / 电影宽屏(2.35:1)

#### 第五步：生成与下载
1. 点击「立即生成」
2. 等待生成完成（通常 1-3 分钟）
3. 预览效果，可下载或重新生成

### 高级功能入口

#### 视频延长
1. 在已生成视频下方点击「延长视频」
2. 输入延长描述
3. **注意**：生成长度应选择「新增部分」的时长

#### 首尾帧生成
1. 上传首帧图片（必选）
2. 可选上传尾帧图片
3. 描述中间过渡内容

#### 多模态参考

## 📝 Seedance 2.0 提示词优化指南

### 提示词结构模板

```
【整体描述】风格 + 时长 + 画面比例 + 整体氛围

【分镜描述】
0-X秒：[镜头运动]，[画面内容]，[主体动作]，[光影/特效]
X-Y秒：[镜头运动]，[画面内容]，[主体动作]，[光影/特效]
...

【声音说明】配乐风格/音效/对白（如有音频参考可标注 @音频1）

【参考素材说明】（如有上传）
@图片1 作为首帧/角色参考
@视频1 参考运镜/动作
@音频1 用于配乐/对白参考
```

### 提示词书写原则

1. **时间轴清晰** - 标注每段的秒数范围（0-3秒、3-6秒等）
2. **镜头语言明确** - 使用专业术语（推/拉/摇/移/跟/环绕/升降/希区柯克变焦）
3. **动作描述具体** - 避免模糊词汇（"做一些动作"），使用具体动作描述
4. **多模态引用** - 使用 `@素材名` 格式引用上传的文件
5. **延长视频说明** - 需要延长时说明"延长Xs"并选择对应生成长度

### 专业运镜词汇表

| 运镜类型 | 说明 | 适用场景 |
|---------|------|----------|
| **推** | 镜头向前推进，主体逐渐放大 | 强调主体、制造紧张感 |
| **拉** | 镜头向后拉远，视野逐渐扩大 | 展现场景、营造氛围 |
| **摇** | 镜头固定位置，左右或上下转动 | 展示环境、跟随主体 |
| **移** | 镜头平行移动 | 跟随运动主体、展现场景 |
| **跟** | 镜头跟随主体移动 | 保持主体在画面中 |
| **环绕** | 镜头围绕主体旋转 | 展示立体造型、增强视觉冲击 |
| **升降** | 镜头垂直上下移动 | 展现场景层次、制造氛围 |
| **希区柯克变焦** | 变焦同时反向移动，背景变化主体不变 | 制造紧张、眩晕感 |

### 风格关键词库

| 风格类型 | 关键词 |
|---------|--------|
| **写实类** | 电影级写实、纪录片风格、自然光影、电影感 |
| **动画类** | 3D动画、二次元、手绘风格、定格动画、像素风 |
| **艺术风格** | 水墨画、油画、水彩、素描、版画、工笔画 |
| **电影风格** | 赛博朋克、蒸汽朋克、废土风、复古胶片、 noir 黑色电影 |
| **色调氛围** | 暖色调、冷色调、高对比度、低饱和度、霓虹光、自然光 |

## 💡 分步引导流程

### 第一步：理解用户想法

用友好的方式询问用户：

1. **你想讲什么故事？**（一句话概括核心内容）
2. **视频时长？**（4-15秒，默认15秒）
3. **参考素材情况？**（有无可用的图片/视频/音频参考）

### 第二步：深入挖掘细节

根据用户的初步回答，逐步引导完善以下维度：

#### 1. 内容叙事
- 故事的起承转合是什么？
- 关键情节有哪些？
- 需要哪些角色/主体？
- 对白/旁白内容？

#### 2. 视觉风格
- 整体风格：写实/动画/水墨/科幻/复古/电影感？
- 色调氛围：明亮/昏暗/暖色/冷色/黑白？
- 画面比例：竖屏(9:16)/横屏(16:9)/电影宽屏(2.35:1)？

#### 3. 镜头语言
- 景别变化：远景→全景→中景→近景→特写？
- 运镜方式：推/拉/摇/移/跟/环绕/升降/希区柯克变焦？
- 转场方式：硬切/渐变/匹配剪辑/特效转场？

#### 4. 动作与节奏
- 主体动作：行走/奔跑/打斗/舞蹈/表情变化？
- 动作节奏：舒缓/急促/有张有弛？
- 是否需要音乐卡点？

#### 5. 声音设计
- 配乐风格：激昂/温馨/悬疑/欢快/史诗？
- 音效需求：环境音/动作音效/特殊音效？
- 对白/旁白：有台词吗？什么语气？

### 第三步：构建分镜结构

将故事拆解为时间轴上的镜头序列：

```
示例结构（15秒视频）：
0-3秒：开场镜头，建立场景
3-6秒：发展，引入主体/冲突
6-10秒：高潮，核心动作/情感爆发
10-13秒：转折/过渡
13-15秒：结尾/落版
```

### 第四步：生成专业提示词

根据收集的信息，生成符合 Seedance 2.0 语法的提示词（见上文模板）。

### 第五步：优化与建议

- 检查是否违反平台规则（如写实人脸限制）
- 提供多模态输入建议（什么素材最有参考价值）
- 给出替代方案（如无法实现时的调整建议）

## 📚 高级技巧与实战

### Seedance 2.0 新特性应用

#### 1. 多模态深度融合

**场景**：想要精确控制角色形象和动作

**技巧**：
```
@图片1 作为角色形象参考（正面全身照）
@图片2 作为角色表情参考（特写）
@视频1 作为动作节奏参考
@音频1 作为配乐参考
```

#### 2. 复杂运镜复刻

**场景**：想要电影级的镜头语言

**技巧**：
```
希区柯克变焦：镜头推进同时变焦，背景压缩，制造紧张感
环绕镜头：360° 展示主体，增强立体感
升降镜头：从低角度升到高角度，展现场景层次
跟拍镜头：跟随运动主体，保持动感
```

#### 3. 视频延长创作

**场景**：15秒不够，需要更长视频

**技巧**：
1. 先生成 15 秒基础视频
2. 使用「延长视频」功能
3. 描述接下来的剧情
4. **关键**：生成长度选择「新增部分」的时长，不是总时长

#### 4. 首尾帧生成

**场景**：精确控制视频开始和结束画面

**技巧**：
```
上传首帧图片（必选）：视频第一帧画面
上传尾帧图片（可选）：视频最后一帧画面

提示词：从首帧画面开始，[描述中间过渡内容]，最终到达尾帧画面
```

### 特殊场景解决方案

| 场景 | 问题 | 解决方案 |
|------|------|----------|
| **角色不一致** | 同角色在不同镜头中样子不同 | 上传 3-5 张同一角色的多角度参考图 |
| **动作不流畅** | 动作生硬或不符合物理规律 | 提供参考视频展示期望的动作节奏 |
| **场景跳变** | 场景突然变化，缺乏过渡 | 使用时间轴描述，确保场景转换有镜头过渡 |
| **风格不稳定** | 视频前后风格不一致 | 在提示词开头明确整体风格关键词 |
| **时长不够** | 15秒无法完整表达 | 分多次生成，使用视频延长功能 |
